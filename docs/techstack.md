Frontend
React – Component-based UI for building the interactive grid interface
Vite – Fast development server and optimized production builds
TypeScript – Type-safe frontend and backend development
Tailwind CSS – Utility-first styling for clean and responsive UI
shadcn - for UI componments
Framer Motion – Smooth tile claim animations and micro-interactions
Zustand – Lightweight client state management for tiles, users, and UI state

Backend
Node.js – Backend runtime environment
Express – REST API handling and server structure
Socket.IO – WebSocket-based real-time communication for instant tile updates
TypeScript – Shared typing and maintainable backend code
node-postgres – Direct SQL queries and connection pooling

Database & Caching
PostgreSQL – Persistent storage for users, tiles, sessions, and claim history
Redis – Fast caching, tile locking, cooldowns, and leaderboard counters
Deployment / DevOps
Render – Backend hosting
Vercel / Netlify – Frontend deployment
Docker – Consistent local development and deployment environments
GitHub – Version control and public repository


Why This Stack
- Fast frontend development with Vite
- Smooth UI with React + Tailwind
- Reliable real-time updates with Socket.IO
- PostgreSQL for durable data storage
- Direct SQL gives more control over performance-sensitive queries
- Redis reduces DB load and handles concurrency safely
- TypeScript across the stack for maintainability
- Easy deployment on Render/Vercel
